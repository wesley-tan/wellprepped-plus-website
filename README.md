# WellPrepped+ Website

A modern, responsive website for WellPrepped+, a premier IB education service. Built with Next.js, React, TypeScript, and Tailwind CSS.

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd wellprepped-website
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   ```

4. **Open your browser**
   Navigate to [http://localhost:3000](http://localhost:3000)

## 📝 Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run start` - Start production server
- `npm run lint` - Run ESLint

## 🎨 Design System

### Colors
- **Primary Background**: `#1d231c`
- **Button Background**: `#283b28`
- **Text**: White and gray variants
- **Accent**: Green gradients

### Typography
- **Primary Font**: Inter (Google Fonts)
- **Display Font**: Host Grotesk (fallback to Inter)
- **Font Weights**: 400, 500, 600, 700

### Components
- **Buttons**: Rounded with hover effects
- **Cards**: Glass effect with backdrop blur
- **Sections**: Consistent padding and spacing

## 📱 Responsive Breakpoints

- **Mobile**: < 768px
- **Tablet**: 768px - 1024px
- **Desktop**: > 1024px
